# Inclusive Learning System

A Django-based web application for inclusive education, featuring course management, live transcription, notifications, and user authentication.

## Features
- User registration and login
- Course listing and details
- Live transcription tool
- Notifications
- Password recovery

## Setup
1. Clone the repository
2. Create/activate a virtual environment
3. Install dependencies: `pip install -r requirements.txt`
4. Configure environment variables (DJANGO_SECRET_KEY, etc.)
5. Run migrations: `python manage.py migrate`
6. Start the server: `python manage.py runserver`
