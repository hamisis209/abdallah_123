# Accessibility Statement

Inclusive Learning System is committed to providing a website that is accessible to the widest possible audience, regardless of technology or ability.

## Accessibility Features
- Use of ARIA landmarks and roles
- Responsive design for all devices
- Alt text for images and icons
- Keyboard navigation support

## Feedback
If you experience any difficulty accessing content, please contact us at admin@yourdomain.com.
